# Modelagem de Ameaças
Este write-up documenta o processo de identificar, analisar e compreender as ameaças e ações que podem ser tomadas para eliminá-las. 

### Informações do Homelab
- **Projeto**: Homelab - Servidor Seguro
- **Hardware**: Notebook antigo 
- **Sistema Operacional:** Ubuntu Server 24.04 LTS
- **Gerenciador de Serviços:** CasaOS
- **Desenvolvedor:** Ian Sampaio
- **Descrição:** O homelab é um servidor doméstico que hospeda serviços pessoais dos usuários. Sua principal função é servir como nuvem, guardando muitas informações pessoais sensíveis. Por estar em início de desenvolvimento, o servidor está limitado à um usuário (administrador).
